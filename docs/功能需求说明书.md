# 功能需求说明书

## 1. 系统角色
- **未登录用户**：可浏览基金、公司、经理信息，查看画像，但不能保存/管理组合
- **已登录用户**：可使用全部功能，包括组合管理、AI 对话

## 2. 核心功能模块

### 2.1 基金查询模块
- 按基金代码或名称模糊搜索
- 多选标签筛选（同维度取并集，跨维度取交集）
- 搜索结果分页显示（每页 20 条）
- 列表展示基金代码、名称、类型、单位净值、日涨跌幅
- 勾选基金后保存为自定义组合（需登录）

### 2.2 基金公司查询模块
- 按公司名称模糊搜索
- 公司维度标签筛选
- 查看公司旗下基金列表，并支持保存为组合

### 2.3 基金经理查询模块
- 按经理姓名模糊搜索
- 经理维度标签筛选
- 查看经理管理的基金列表，并支持保存为组合

### 2.4 基金画像模块
- 基本信息展示（代码、全称、类型、规模、成立日期、风险等级、跟踪指数、公司、经理）
- 净值走势折线图（近1月/3月/6月/1年可切换）
- 行业配置饼图 + 前十大重仓股表格
- 业绩归因文本
- 最近5条公告列表

### 2.5 组合管理模块
- 查看已保存的组合列表
- 组合详情（包含基金基本信息）
- 删除组合
- 批量操作模式（多选删除）

### 2.6 用户模块
- 注册（校验非空）
- 登录（验证用户名密码）
- 个人空间（修改昵称、修改密码）

### 2.7 AI 智能问答
- 全局悬浮聊天入口
- 调用后端代理接口，基于 DeepSeek 模型回答基金问题
- 未登录点击时提示“请先登录”

### 2.8 鸿蒙APP
- 提供单基金净值查询接口 `GET /api/funds/code/{code}`
